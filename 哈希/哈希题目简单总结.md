**快乐数**
sum = array.reduce((pre,cur)=>pre+cur*cur,0)

pre是之前累积的总和

我写的
```javaScript
var isHappy = function(n) {
let set = new Set()
let array = n.toString().split('')
let sum = 0
while(sum!=1){
   sum = array.reduce((pre,cur)=>pre+cur*cur,0)
   if(set.has(sum)){
       return false
   }
   else{
       set.add(sum)
       array = sum.toString().split('')
   }
}
return true
};
```
