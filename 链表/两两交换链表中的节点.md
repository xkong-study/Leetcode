初始链表：

```
rustCopy code
1 -> 2 -> 3 -> 4
```

在第一次递归调用中，`head` 指向 `1`，`newHead` 指向 `2`。执行 `newHead.next = head;` 后：

```
rustCopy code
1    2 -> 3 -> 4
^    |
|____|
```

接下来，递归调用 `head.next = dfs(newHead.next)` 会处理 `3` 和 `4`。这会导致 `3` 指向 `4` 的下一个节点（如果存在的话），并且 `4` 指向 `3`。所以，当递归调用返回时，节点 `1` 的 `next` 指针会指向节点 `4`：

```
markdownCopy code
1         3    4
^         ^    |
|_________|____|
```
newHead.next = head

最终的链表结构如下：
```
rustCopy code
2 -> 1 -> 4 -> 3
```
